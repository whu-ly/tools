# 批量数据解算与解码工具

Visual Studio 2019 C++ 项目，支持批量数据文件的自动解算或解码，适用于多设备、多数据集的批处理场景。

## 功能说明

- 支持批量数据文件的线性或多线程处理
- 支持两种任务模式：数据转换（TaskType=1）和数据解算（TaskType=2）
- 支持基于文件列表或数据集目录的文件查找处理
- 支持指定设备类型和子数据集进行有针对性的处理
- 线程数可配置，支持并行处理以提高效率

## 使用方法

1. 修改源码中 `main.cpp` 的相关路径和参数配置，包括：
   - `TaskType`（1-转换，2-解算）
   - `ProcType`（处理方式与线程模式）
   - 数据集路径 `m_DataSetPath`
   - 设备类型 `m_Equip` 和子数据集 `m_SubDataSetFolderName`
   - 可执行程序路径 `m_TaskExe` 及相关文件夹、后缀设置
2. 在 Visual Studio 2019 中打开工程并编译
3. 运行生成的可执行文件进行批处理

## 依赖环境

- Windows 操作系统
- Visual Studio 2019 及支持的 C++ 编译环境
